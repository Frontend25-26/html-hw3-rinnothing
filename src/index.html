Какие элементы затронет селектор '~'? 
html:
        <p>1</p>
        <a class="friends" href="https://google.com"></a>
        <q>That's it!</q>
    css:
        .friends ~ * {
            color: red;
        }



`~` затрагивает только элементы того же уровня, находящиеся ниже выбранного первым условием элемента

Какую width покажет inspect element в браузере? 
html:
        <p class="strange"></p>
    css:
        .strange {
            box-sizing: border-box;
            margin: 10px;
            border: solid 5px;
            width: 30px;
        }




при border-box padding и border "уже учтены" в width

Отличия между margin и padding:



margin - снаружи, padding - внутри

visibility:hidden и display:none



visibility:hidden просто делает элемент невидимым, а display:none - даже не рендерит, оба не могут быть прочитаны скринридерами
