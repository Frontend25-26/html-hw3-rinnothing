Какие элементы затронет селектор '~'? 

html:
    <p>1</p>
    <a class="friends" href="https://google.com"></a>
    <q>That's it!</q>
css:
    friends ~ * {
        color: red;
    }



`~` затрагивает только элементы того же уровня, находящиеся ниже выбранного первым условием элемента

Чему равна общая width блока? 

html:
    <p class="strange"></p>
css:
    .strange {
        box-sizing: border-box;
        margin: 10px;
        padding: 20px;
        border: solid 5px;
        width: 30px;
    }




при border-box общая width = margin + width
